Tool yang digunakan adalah Android Studio API 23 dengan google map, google service, google place dan google direction.

Pada dasarnya aplikasi ini terbagi dua:
Aplikasi di client yang berupa file APK yang didownload dan diinstal di HP driver ataupun pengguna jasa.
Aplikasi server yang diinstal diserver untuk menyimpan informasi driver, pelanggan, seting harga per km, driver yang berada diposisi tersebut, pemilihan driver dan peringkat dan lain-lain.
